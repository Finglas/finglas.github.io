Recommended Reading 2012

Published: Mon 01 October 2012

tags: reading

Here is my updated reading list for 2012. We might have missed something so if you think we should be reading it, let us know.

Process

  • The Toyota Way - Jeffrey Liker
  • The Goal - Eli Goldratt
  • XP Explained- Kent Beck
  • The Pragmatic Programmer- Andrew Hunt and David Thomas
  • The Clean Coder - Robert C. Martin

Code

  • Clean Code - Robert C. Martin
  • Test-Driven Development by Kent Beck
  • Refactoring - Martin Fowler
  • Refactoring Workbook - William Wake
  • Patterns of Enterprise Application Architecture - Martin Fowler
  • Head First Design Patterns - Freeman, et al
  • Growing Object-Oriented Software, Guided by Tests - Steve Freeman and Nat Pryce
  • 97 Things Every Programmer Should Know- Kevlin Henney
  • Pro ASP.NET MVC x+ - Steve Sanderson
  • Framework Design Guidelines - Brad Abrams and Krzysztof Cwalina
